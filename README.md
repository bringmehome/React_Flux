# Flux TodoMVC Example

>1、下载zip文件“https://github.com/facebook/flux” ，整个下载

>2、解压后得到 flux-master.zip\flux-master\examples\flux-todomvc里面的文件

>3、整个拷贝到一个文件夹下，执行 npm install，这下就可以直接运行index.html文件了

##至此，flux环境搭建成
